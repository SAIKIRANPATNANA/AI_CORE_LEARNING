# AI Core Learning

A curated learning repository documenting my hands-on journey through the core foundations of **Machine Learning** and **Deep Learning**.

This repository brings together the notebooks, experiments, datasets, and concept-driven practice material I have used to build a stronger understanding of modern AI fundamentals. Rather than acting as a single end-to-end project, it serves as a structured learning track focused on the concepts, implementations, and experiments that sit underneath stronger real-world AI systems.

## What This Repository Covers

- **Machine Learning foundations** through EDA, feature engineering, preprocessing, classical algorithms, and model evaluation
- **Deep Learning foundations** through neural network concepts, PyTorch practice, CNN architectures, transfer learning, regularization, and optimization
- **Implementation-driven learning** through notebooks that turn core theory into hands-on understanding
- **AI fundamentals** that support later work in GenAI, computer vision, NLP, and applied ML projects

## Learning Tracks Inside This Repo

### Machine Learning Learning Track
The `ML_LEARNING` section focuses on practical understanding of classical machine learning workflows, including:
- preprocessing and feature engineering
- regression and classification
- clustering and anomaly detection
- model training and evaluation
- EDA across multiple datasets

Representative topics include:
- missing value handling
- encoding techniques
- feature scaling
- logistic regression
- SVM, KNN, decision trees, random forests, and Naive Bayes
- PCA, DBSCAN, hierarchical clustering, and anomaly detection

### Deep Learning Learning Track
The `DL_LEARNING` section focuses on neural-network-oriented learning and implementation, including:
- perceptrons and network basics
- PyTorch foundations
- activation functions and loss functions
- regularization and optimization
- transfer learning
- CNN foundations and major architectures
- object detection and YOLO-related exploration

Representative topics include:
- AlexNet
- LeNet
- VGG
- ResNet50
- Inception
- RCNN / Fast RCNN / Faster RCNN
- SSD
- YOLO and custom training practice
- Siamese networks
